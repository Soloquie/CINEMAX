# 🎬 CINEMAX

Sistema de gestión para procesos automatizados de un cine.

## 📌 Descripción

CINEMAX es una aplicación  desarrollada para administrar la operación de un cine, incluyendo la gestión de funciones, gestión de confitería y compra de boletas.

El proyecto está organizado por capas, lo que facilita su mantenimiento, escalabilidad y comprensión.

---

## 🚀 Funcionalidades principales

- Gestión de funciones
- Catálogo de funciones disponibles
- Selección de asientos
- Carrito de compra
- Registro e inicio de sesión de usuarios
- Seguridad con JWT
- Recuperación y verificación de cuenta por correo
- Carga de imágenes con Cloudinary

---

## 🛠️ Tecnologías utilizadas

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- Maven
- MySQL
- JWT
- SendGrid
- Cloudinary
- Lombok
